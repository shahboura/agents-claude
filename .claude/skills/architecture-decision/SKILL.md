Create an ADR for: $ARGUMENTS

Structure the document with these required sections:
1. **Context** — what forces and constraints make this decision necessary now
2. **Options considered** — at least 2 alternatives with a brief description each
3. **Tradeoffs** — pros/cons table or comparison for each option
4. **Decision** — the chosen option and the primary reason
5. **Consequences** — positive outcomes, negative tradeoffs accepted, and follow-up actions required
